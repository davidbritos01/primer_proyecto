Por qué un blog ?

Principalmente es un espacio donde voy a compartir ideas y noticias del mundo GNU/linux, o sea, nada que no se pueda encontrar googleando un poco, al igual que todo lo que voy a escribir.
Y como este este año comencé a dar mis primeros pasos en el mundo del Desarrollo Web, qué mejor que mostrar como instalar el famoso y popular Visual Studio Code en distribuciones basadas en Debian, como Ubuntu, Linux Mint y un sin fin de etcétera.
Seguramente los linuxeros puristas, van exclamar con los teclados en alto, que se puede programar o tirar código en un editor de texto como Vi,Vim, Nano etc., lo cual es cierto, pero desde mi punto de vista, lo apropiado para un junior es empezar por lo básico.

Para descargar Visual Studio Code, se puede hacer utilizando linea de comandos o bien a través de nuestro navegador favorito (Firefox, Google Chrome, Brave etc.)

La decarga es medianamente accesible, pero para la instalación es necesario utilizar la famosa Terminal e ingresar un comando para dar inicio a la instlación de Visual Studio Code.
Pero antes de dar este salto, es necesario saber dónde estamos posicionados para saber ubicar nuestro directorio de descargas.
Por defecto o de manera predeterminada si escribimos el comando pwd y luego presionamos enter, nos va mostrar por pantalla nuestra ubicación actual. Lo siguiente es utilizar cd, lo cual no permite movernos/desplazarnos entre diferentes directorios de nuestro sistema operativo, pero en este caso, el que nos interesa saber, es el de Descargas. Lo último, es ultilizar el comando dpkg -i code_1.81.1-1691620686_amd64.deb, ingresar nuestra password o contraseña de usuario y un último enter. A disfrutar Visual Studio Code.

Y bue sino se entendió dejó un link con un video para chequear estos pasos

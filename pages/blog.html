Visual Studio Code en Debian

Visual Studio Code está en el top 10 de los editores para desarrollo más utilizado. Este popular editor el cual fue creado por Microsfot, está disponible para Windows, macOS y Linux de manera gratuita.

Si nos centramos en Linux como lugar de interés, es bastante sencillo su instalación. Para este ejemplo se va explicar cómo hacerlo en Debian, pero también aplica estos pasos para Ubuntu, Mint etc.

Para descargarVisual Studio Code, se puede hacer utilizando la famosa línea de comandos o bien a través de nuestro navegador favorito (Firefox, Google Chrome, Brave etc.)

La decarga es medianamente accesible, pero para la instalación es necesario utilizar la famosa Terminal e ingresar un comando para dar inicio a la instlación de Visual Studio Code. Pero antes de dar este salto, es necesario saber dónde estamos posicionados para saber ubicar nuestro directorio de descargas.
Con el comando pwd nos va mostrar por pantalla en qué directorio nos encontramos posicionado. Lo siguiente es utilizar el comando cd, lo cual no va permitir movernos/desplazarnos entre los distindos directorios de nuestro sistema operativo, pero en este caso, el que nos interesa saber, es el de Descargas. Lo último, es ultilizar el comando dpkg -i code_1.81.1-1691620686_amd64.deb, ingresar nuestra password/contraseña de usuario y un último enter.
A disfrutar Visual Studio Code !!.

En el caso hipotético y/o posible de que no puedas instalar VS Code en tu PC, te dejó abajo un video para chequear estos pasos. La verdad que está bastante bueno y permite resumir en pocos pasos todo lo explicado.
